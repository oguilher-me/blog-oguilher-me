Observação: Se adicionarmos essa chamada antes de inicializar os componentes do JFrame causará um erro por que não temos componentes inicializados para adicionar o efeito. Portanto, essa chamada deve ser realizada EXCLUSIVAMENTE após a invocação do initComponents()
